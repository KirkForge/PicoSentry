# bun-runtime-helper

A trivial helper for projects that target the Bun JavaScript runtime.
Targets the [Bun](https://bun.sh) runtime and falls back to Node.js
when Bun is not available.

This package has no postinstall hooks, no native dependencies, and
ships only a single source file. It contains no known campaign
indicators and no payload filenames from any tracked supply-chain
attack corpus.
